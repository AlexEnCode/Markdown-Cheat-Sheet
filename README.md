## Headers
```markdown
# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6
```

---

## Bold and Italic Text
```markdown
**Bold** or __Bold__
*Italic* or _Italic_
***Bold and Italic*** or ___Bold and Italic___
```

---

## Lists
### Unordered List
```markdown
- Item 1
- Item 2
  - Subitem
    - Sub-subitem
```

### Ordered List
```markdown
1. Item 1
2. Item 2
   1. Subitem
   2. Subitem
```

---

## Links and Images
### Link
```markdown
[Link Text](https://example.com)
```

### Image
```markdown
![Alt Text](https://example.com/image.jpg)
```

---

## Blockquotes
```markdown
> This is a blockquote.
```

---

## Code
### Inline Code
```markdown
Here is `inline code`.
```
